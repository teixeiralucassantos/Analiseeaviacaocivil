# Análise de Ocorrências Aeronáuticas na Aviação Civil Brasileira

Este projeto em Python utiliza SQLite para criar uma base de dados sobre a aviação civil no Brasil, coletando dados relevantes e gerando gráficos para análise. O resultado final é um dashboard interativo em HTML que visualiza as informações de maneira clara e acessível.

## Descrição

O objetivo deste projeto é coletar dados sobre ocorrências aeronáuticas na aviação civil brasileira, armazená-los em um banco de dados SQLite e apresentar essas informações através de um dashboard interativo. O dashboard inclui gráficos que ilustram as ocorrências, suas classificações e outras informações relevantes.

## Tecnologias Utilizadas

- **Python**: Linguagem de programação utilizada para desenvolvimento.
- **SQLite**: Sistema de gerenciamento de banco de dados para armazenar dados.
- **Pandas**: Biblioteca para manipulação e análise de dados.
- **Matplotlib**: Biblioteca para geração de gráficos.
- **Dash**: Framework para construção de aplicações web interativas em Python.
- **HTML**: Estrutura de apresentação do dashboard.
